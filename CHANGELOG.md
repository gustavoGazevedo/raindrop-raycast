# Raindrop Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Search across all Raindrop.io bookmarks with full-text search
- Filter by collection with hierarchical dropdown
- Filter by bookmark type (link, article, image, video, document, audio)
- Sort by relevance, date, title, or domain
- Infinite scroll pagination
- Open bookmarks in browser or Raindrop.io web app
- Display bookmark metadata: favicon, type, tags, collection, and domain
- Support for advanced search operators (#tag, type:, created:, link:, title:, etc.)
- Keyboard shortcuts for navigation and actions
- Detail panel with bookmark excerpt and metadata
